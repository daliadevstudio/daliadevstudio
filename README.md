## 🔄 Development Workflow

<div align="center">

**Discovery → Planning → Design → Development → Testing → Deployment → Maintenance**

</div>

1. **Discovery** — Deep-dive into your goals, users and constraints.
2. **Planning** — Scope, architecture, timeline, milestones, clearly defined.
3. **Design** — Flows, interactions, and pixel-perfect UI in a live prototype.
4. **Development** — Clean code shipped in tight iterations you can preview daily.
5. **Testing** — QA, performance, accessibility, verified across devices.
6. **Deployment** — Zero-downtime release with monitoring and rollback ready.
7. **Maintenance** — Ongoing improvements, updates, and support that scale with you.

<!--  SETUP GUIDE — GitHub Actions for Snake Animation              -->
<!-- ═══════════════════════════════════════════════════════════════ -->
<details>
<summary>🐍 <b>Setup Guide: Contribution Snake Animation</b></summary>

To enable the animated snake, create this file at `.github/workflows/snake.yml` in your profile repo:

```yaml
name: Generate Snake Animation

on:
  schedule:
    - cron: "0 0 * * *"
  workflow_dispatch:

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: daliadevstudio
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark
      - uses: crazy-max/ghaction-github-pages@v3
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

</details>
